1

Netlify Drop（拖放部署）

app.netlify.com/drop
⚡ 30秒上線 🏆 新手首選 無需帳號 HTTPS自動
💡
完全不需要任何帳號、不需要 Git、不需要寫程式。就是「拖一下」。
  • 打開瀏覽器，前往 app.netlify.com/drop
  • 把你的 cps-auto-system.html 檔案，直接拖進瀏覽器畫面的灰色方框
  • 等待約 5–10 秒，系統自動處理
  • 完成！畫面會顯示你的網址，格式如：random-name.netlify.app
  • 【進階】點「Claim this site」免費申請帳號，可以改成自訂網址如 my-cps-deals.netlify.app
🌐 上線網址範例：https://amazing-deals-tw.netlify.app
⚠️
未登入的臨時部署會在 24小時後自動刪除。免費申請帳號後可永久保留。
2

Tiiny.host（最簡單）

tiiny.host
⚡ 1分鐘上線 極簡操作 免費方案
  • 前往 tiiny.host
  • 輸入你想要的網站名稱（如：my-cps-deals）
  • 點選「Upload file」，選擇你的 HTML 檔案
  • 輸入電子信箱後，點擊「Upload」
  • 收取驗證信，點擊連結確認即完成
🌐 上線網址範例：https://my-cps-deals.tiiny.site
⚠️
免費方案每月流量上限 50,000 次訪問，檔案大小上限 3MB，對初期使用完全足夠。
3

GitHub Pages（工程師標配）

github.com → Settings → Pages
⏱ 5–10分鐘 需要 GitHub 帳號 永久免費 版本控制
  • 前往 github.com 免費申請帳號（如果還沒有的話）
  • 點「New Repository」，取名如 cps-auto，設定為 Public（公開）
  • 點「uploading an existing file」，把 HTML 檔案拖進去，注意：檔名必須改為 index.html
  • 點「Commit changes」儲存
  • 進入 Settings → Pages → Source 選 main branch → Save
  • 等 1–2 分鐘，重新整理頁面即可看到你的網址
🌐 上線網址範例：https://你的用戶名.github.io/cps-auto/
4

Cloudflare Pages（效能最強）

pages.cloudflare.com
⏱ 10–15分鐘 🏆 最佳長期選擇 無限流量 全球CDN
  • 先完成「方法③ GitHub Pages」的步驟，把 index.html 推上 GitHub
  • 前往 pages.cloudflare.com，用 Google 或 Email 免費申請 Cloudflare 帳號
  • 點「Create a project」→「Connect to Git」→ 連結你的 GitHub 帳號
  • 選擇 cps-auto 這個倉庫
  • Framework preset 選 None，Build command 留空，點「Save and Deploy」
  • 等 1–2 分鐘部署完成，即可獲得一個超快的全球 CDN 網址
🌐 上線網址範例：https://cps-auto.pages.dev
🏆
Cloudflare 在台灣有節點，台灣用戶訪問速度極快，而且免費方案有無限流量，是長期運營的最佳選擇。
5

Vercel（Next.js 御用）

vercel.com
⏱ 5–10分鐘 開發者友好 預覽部署 GitHub整合
  • 先完成「方法③」，把 index.html 推上 GitHub
  • 前往 vercel.com，用 GitHub 帳號直接登入
  • 點「New Project」，選取你的 cps-auto GitHub 倉庫
  • Framework Preset 選 Other，所有設定保持預設
  • 點「Deploy」，等 30 秒即完成
  • 往後每次更新 GitHub，Vercel 會自動重新部署
🌐 上線網址範例：https://cps-auto.vercel.app
